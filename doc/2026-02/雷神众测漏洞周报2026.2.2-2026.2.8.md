漏洞详情  
  
**1.Ivanti EPMM存在命令注入漏洞**  
  
  
漏洞介绍：  
  
Ivanti EPMM（端点管理器移动版）是一款企业级移动设备与应用程序管理（MDM/EMM/UEM）解决方案。它专注于帮助企业管理员工移动设备、保障企业数据安全，并安全地分发企业应用程序。其核心功能包括设备合规性监控、应用级数据容器化隔离，以及提供安全的移动生产力工具。  
  
  
漏洞危害：  
  
Ivanti Endpoint Manager Mobile (EPMM)披露存在未授权命令注入漏洞（CVE-2026-1281、CVE-2026-1340）。由于系统核心组件在处理外部输入时存在严重的过滤缺陷，远程且未经身份验证的攻击者可通过构造恶意的 HTTP 请求，在目标服务器上实现任意命令执行。官方确认，该漏洞已在野外被积极利用，攻击目标指向关键基础设施的移动端管控平台。  
  
  
漏洞编号：  
  
CVE-2026-1281  
  
CVE-2026-1340  
  
  
影响范围：  
  
Ivanti Endpoint Manager Mobile <= 12.5.0.0  
  
Ivanti Endpoint Manager Mobile <= 12.5.1.0  
  
Ivanti Endpoint Manager Mobile <= 12.6.0.0  
  
Ivanti Endpoint Manager Mobile <= 12.6.1.0  
  
Ivanti Endpoint Manager Mobile <= 12.7.0.0  
  
  
修复方案：  
  
及时测试并升级到最新版本或升级版本  
  
  
来源:  
安恒信息CERT  
  
**2.大蚂蚁即时通讯系统存在任意文件上传漏洞**  
  
  
漏洞介绍：  
  
大蚂蚁即时通讯系统是一款企业办公通讯软件，通过本地化服务器部署为政企单位提供安全可控的协同办公服务。  
  
  
漏洞危害：  
  
攻击者可通过该文件上传漏洞实现目录穿越上传恶意脚本文件至web目录进而获取服务器权限。  
  
  
影响范围：  
  
BigAnt >= 5.5.x  
  
  
修复方案：  
  
及时测试并升级到最新版本或升级版本  
  
  
来源：  
安恒信息CERT

**4.Apache NiFi代码问题漏洞**  
  
  
漏洞介绍：  
  
Apache NiFi是美国阿帕奇（Apache）基金会的一套数据处理和分发系统。该系统主要用于数据路由、转换和系统中介逻辑。  
  
  
漏洞危害：  
  
Apache NiFi存在代码问题漏洞，该漏洞源于GetAsanaObject Processor使用未过滤的Java对象反序列化，攻击者可利用该漏洞导致执行任意代码。  
  
  
漏洞编号：  
  
CVE-2025-66524  
  
  
影响范围：  
  
Apache Nifi >=1.20.0，<=2.6.0  
  
  
修复方案：  
  
及时测试并升级到最新版本或升级版本  
  
  
来源：  
CNVD
