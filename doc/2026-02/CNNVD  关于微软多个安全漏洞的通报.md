#  CNNVD | 关于微软多个安全漏洞的通报  
 中国信息安全   2026-02-12 08:23  
  
[](https://cisat.cn/all/14915419?navIndex=3)  
  
  
**漏洞情况**  
  
近日，微软官方发布了多个安全漏洞的公告，其中微软产品本身漏洞55个，影响到微软产品的其他厂商漏洞3个。微软Microsoft Windows、Microsoft Hyper-V、Microsoft Windows Kernel、Microsoft MSHTML Framework等多个产品和系统受漏洞影响。目前，微软官方已经发布了漏洞修复补丁，建议用户及时确认是否受到漏洞影响，尽快采取修补措施。  
  
## 一漏洞介绍  
  
  
2026年2月10日，微软发布了2026年2月份安全更新，共58个漏洞的补丁程序，CNNVD对这些漏洞进行了收录。本次更新主要涵盖了Microsoft Windows 和 Windows 组件、Microsoft Hyper-V、Microsoft Windows Kernel、Microsoft MSHTML Framework、Microsoft Word、Microsoft GitHub Copilot for JetBrains等。CNNVD对其危害等级进行了评价，其中超危漏洞2个，高危漏洞39个，中危漏洞16个，低危漏洞1个。  
  
微软多个产品和系统版本受漏洞影响，具体影响范围可访问微软官方网站查询：  
  
https://portal.msrc.microsoft.com/zh-cn/security-guidance  
  
****  
## 二漏洞详情  
  
  
此次更新共58个漏洞的补丁程序，包括54个新增漏洞的补丁程序、1个更新漏洞的补丁程序和3个影响微软产品的其他厂商漏洞的补丁程序。  
  
此次更新共包括54个新增漏洞的补丁程序，其中超危漏洞1个，高危漏洞38个，中危漏洞14个，低危漏洞1个。

## 三修复建议  
  
  
目前，微软官方已经发布补丁修复了上述漏洞，建议用户及时确认漏洞影响，尽快采取修补措施。微软官方补丁下载地址：  
  
https://msrc.microsoft.com/update-guide/en-us  
  
CNNVD将继续跟踪上述漏洞的相关情况，及时发布相关信息。如有需要，可与CNNVD联系。联系方式: cnnvd@itsec.gov.cn
