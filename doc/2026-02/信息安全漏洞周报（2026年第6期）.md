1. Keylime 安全漏洞（CNNVD-202602-963）****  
  
Keylime是一个开源的利用TPM技术的可扩展信任系统。  
  
Keylime 7.12.0及之前版本存在安全漏洞，该漏洞源于未强制要求用户执行客户端TLS身份验证，攻击者利用该漏洞可以获取敏感数据。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/keylime/keylime/releases  
  
2. Wing FTP Server 代码问题漏洞（CNNVD-202602-893）****  
  
Wing FTP Server是一个开源的跨平台的FTP服务软件。  
  
Wing FTP Server 6.0.7版本存在代码问题漏洞，该漏洞源于服务路径中未添加引号，攻击者利用该漏洞可以提升权限并执行任意代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://www.wftpserver.com/download.htm  
  
3. WordPress plugin LatePoint 跨站脚本漏洞（CNNVD-202602-485）****  
  
WordPress和WordPress plugin都是WordPress基金会的产品。WordPress是一套使用PHP语言开发的博客平台。该平台具有在基于PHP和MySQL的服务器上架设个人博客网站的功能。WordPress plugin LatePoint是一个应用插件。  
  
WordPress plugin LatePoint 5.2.5及之前版本存在跨站脚本漏洞，该漏洞源于对用户的输入清理和输出转义不足，攻击者利用该漏洞可以注入任意Web脚本。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://wordpress.org/plugins/latepoint

1. Lunary 安全漏洞（CNNVD-202602-118）****  
  
Lunary是一个开源的LLM开发工具平台。  
  
Lunary 1.2.2版本存在安全漏洞，该漏洞源于对密码重置令牌的安全防护不足，攻击者利用该漏洞可以获取其他用户的密码并重置令牌。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://lunary.ai/  
  
2.Claude Code 操作系统命令注入漏洞（CNNVD-202602-978）  
  
Claude Code是一个开源的终端原生AI编程工具。  
  
Claude Code 2.0.55之前版本存在操作系统命令注入漏洞，该漏洞源于未能正确验证使用带echo命令的管道sed操作命令，攻击者利用该漏洞可以将任意文件写入敏感目录中。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/anthropics/claude-code/tags  
  
3. MLflow 安全漏洞（CNNVD-202602-117）****  
  
MLflow是一个开源的简化机器学习开发平台，具有跟踪实验、将代码打包成可重复运行及共享和部署模型的功能。  
  
MLflow 2.20.3版本存在安全漏洞，该漏洞源于为创建Python虚拟环境分配的临时目录权限限制不当，攻击者利用该漏洞可以覆盖虚拟环境中的.py文件，从而执行任意代码。  
  
目前厂商已发布升级补丁以修复漏洞，参考链接：  
  
https://github.com/mlflow/mlflow/releases
