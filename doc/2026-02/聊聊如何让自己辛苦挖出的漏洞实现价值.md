#  聊聊如何让自己辛苦挖出的漏洞实现价值  
原创 允星河
                    允星河  Joker One Security   2026-02-12 09:47  
  
本篇仅个人看法和个人方法，仅针对当前的形势来进行的探索性调整，不正之处还请各位师傅海涵  
  
  
    大家可以适当把精力放到自动化上，并不是让大家去当脚本小子，而是将时间碎片集中化。  
  
  
    拿到项目第一件事是评估下需要投入多少精力和头发，其次再考虑项目来源是否稳定结款，人为因素导致的“忽略”和“抱歉”是必然会发生的，那么我们需要关注的方向就从某SRC平台慢慢变成某初创公司的某项目，我这么说并不是要支持某初创公司，而是初创公司会为了公司早期的可持续发展性得到一些具有潜力的项目，换句话来说就是初创公司有更大和更稳定支付的前期能力。  
  
  
    拿到项目后第二件事就是要了解规模体系，也就是我们嘴里常说的“信息收集”这里不给大家赘述了，项目开始后要尽快了解到甲方那边注重什么样的漏洞，也就是学会探“口味”，不是为了迎合，而是为了在同批次人群里超车，如果能力足够强，鬼内交易到源码去砍0day那种大佬除外，初学者或者正在爬山的师傅们可以尝试拳打百遍的套路，我会什么 我就死抠什么，抠不到没关系，学习下别人的思路和技巧方式。
