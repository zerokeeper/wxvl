#  大疆扫地机器人被曝远程控制漏洞影响全球设备  
ICICS
                    ICICS  工业信息安全创新中心   2026-03-10 08:28  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_gif/x3IX43J5RG64YpU9k5z05wO7h55bSkl5IuaTRU8gkk3vCw2X8ic39mGWQjJiccvo0k78l0ic6caNYzXqh6Enyom4w/640?from=appmsg "")  
  
  
  
安全研究人员披露，大疆部分扫地机器人存在远程非法控制漏洞，全球约6700台设备受到影响。攻击者可能利用该漏洞绕过身份认证，通过网络远程控制设备执行操作。  
  
该漏洞涉及设备远程管理接口的安全机制缺陷，使得攻击者在特定条件下能够接管设备控制权限。被利用后，攻击者可能操控设备运行、获取设备数据甚至利用设备进入家庭网络。  
  
随着智能家居设备快速普及，扫地机器人、智能摄像头等设备普遍具备联网和远程管理功能，一旦安全设计不足，就可能成为网络攻击的入口。  
  
厂商通常需要通过软件更新或固件升级修复相关漏洞，同时加强设备安全设计，包括身份认证机制、通信加密和访问控制等。该事件再次表明物联网设备安全已成为网络安全治理的重要领域，厂商和用户都需要提高安全意识并及时更新设备系统。
