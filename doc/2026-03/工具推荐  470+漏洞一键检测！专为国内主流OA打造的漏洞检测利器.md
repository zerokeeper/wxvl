**背景介绍**  
  
本次发布为 **I-Wanna-Get-All 工具最终版**  
，共集成 **470 个漏洞检测与利用模块**  
，主要更新内容如下：  
  
**一、新增模块**  
- 新增 **List、JavaMSGenerator、Ysoserial、JeecgReportDatabase**  
 等模块，进一步扩展漏洞覆盖与利用能力。  
  
**二、检测方式优化**  
- 针对用友部分序列化反序列化漏洞，优化检测逻辑，支持 **DNSLog 与 CmdEcho 双模式回显验证**  
，提升无回显漏洞的检测可靠性。  
  
**三、内置 DNSLog 联动机制**  
- 集成内置 DNSLog 平台调用接口，默认使用 http://www.dnslog.cn  
。  
  
- 使用前自动初始化 DNSLog 地址，若初始化失败或未获取到地址，请检查网络连通性，建议配置 DNS 为 223.5.5.5  
。  
  
**四、HTTP 请求配置优化**  
- 优化全局 HTTP 请求处理逻辑，增强适配性。  
  
- 使用 Jeecg 相关漏洞检测时，需在请求中指定有效的 **Token 请求头**  
。  
  
**五、漏洞来源与实现说明**  
- 所有漏洞细节均为个人收集并实现 EXP，审计方式包括：  
  
- 补丁对比分析  
  
- 公开代码审计文章  
  
- 已知 PoC/Exp 复现与改进  
  
- 代码未做混淆，可供安全研究人员审阅与二次开发。  
  
**六、功能集成与研究成果**  
- 集成并二次开发多个知名安全项目模块，融合个人在漏洞利用与绕过方面的研究成果。  
  
- 所有利用模块均在本地环境中测试通过，确保稳定可用。
