# 资产收集  
  
第一步是收集资产。某国际组织作为一个庞大的政治经济实体，其下辖机构众多，域名 xxx  
 覆盖的范围极大。为了覆盖尽可能多的资产，直接去小黄鱼租用一天的 Fofa API 权限，利用网络测绘数据筛选目标。  
  
![QQ_1772246986373.png](https://mmbiz.qpic.cn/mmbiz_png/mcko8AHj6QWV99n1WBwsVklAC6b0tQcckxSHAA588xR3GrMibNG1P0aNGF5x2aacMn3uTz7jr4ZsS76eOsHJ7PmC5ibSIvKfh8w0MyDZOxA3I/640?wx_fmt=png&from=appmsg "")  
  
搜索子域名 xxx  
 并导出数据。Fofa 的一万条导出限制对于此目标已经足够使用，同时也有助于发现未被 DNS 记录收录的资产。  
  
随后将所有 URL 导入辅助工具进行分拣，去重并剔除无法访问的目标后，最终获得四千多个不重复的 URL。  
  
![QQ_1772247006951.png](https://mmbiz.qpic.cn/mmbiz_png/mcko8AHj6QX5rUagriazUSnVemWVzMIQVE5WNxW7jOBsHSXOq9dPibvVSLJHex6Jpfg2tCORCnZcQNj1Bgd4H8Qwns7FbhwNwDzKzI4F1XCPE/640?wx_fmt=png&from=appmsg "")  
  
接着使用 Web 指纹识别工具扫描这些 URL。通过识别技术栈，可以针对性地寻找已被公开披露的漏洞。扫描时开启了 POC 检测和主动指纹识别功能。  
  
![QQ_1772247015210.png](https://mmbiz.qpic.cn/sz_mmbiz_png/mcko8AHj6QUFIDgvzyVic7XH6YbYX3yl7nCKhudoRfhoibuPx2eFEtJdfic4z37uKF5pehm4AVxtMgibSLhJicq1EHR00lTFYKKE0giaCdYZcvoP0/640?wx_fmt=png&from=appmsg "")

### Jira Token 泄露  
  
目标站点为 https://xxx 。该站点目前已无法访问。  
  
前端代码中直接硬编码了 Jira 的 Token。  
  
![QQ_1772247040373.png](https://mmbiz.qpic.cn/mmbiz_png/mcko8AHj6QVHQwytsYhxdVKx8axcKpzEfU1Ot5DlHibEBtibY7kplNAS0kUYGrqSBlicEk2ibCbF2kR57JxB93TENdt2vFXQYl2iacUkZj5IvbR8/640?wx_fmt=png&from=appmsg "")  
  
通过该 Token，翻找 Jira 评论，在备注中发现了某台机器的 Cloud-Init 配置文件。  
```
#cloud-configusers:  - name: vmuser    passwd: "$6$rounds=4096$SWVubDhpuFjRd6zH$028.pRwZD9Bp8lQD3vqA9IGEMvDdGLISEWN2jaounU.hJP31XN4.gHMKCizB6hDp3Frt4xu6AkgNQPDEL1Cl4/"    shell: /bin/bash    lock_passwd: false    ssh_pwauth: True    sudo: ""    groups: users  - name: vmadmin    passwd: "$6$rounds=4096$SWVubDhpuFjRd6zH$028.pRwZD9Bp8lQD3vqA9IGEMvDdGLISEWN2jaounU.hJP31XN4.gHMKCizB6hDp3Frt4xu6AkgNQPDEL1Cl4/"    shell: /bin/bash    lock_passwd: false    ssh_pwauth: True    sudo: ALL=(ALL) NOPASSWD:ALL    groups: users, admin
```  
