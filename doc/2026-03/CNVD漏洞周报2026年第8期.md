1、Adobe产品安全漏洞  
  
Adobe InDesign是由Adobe公司开发的一款专业桌面出版软件，主要用于印刷品和数字媒体的版面设计与页面排版。Adobe Dreamweaver Desktop是美国奥多比（Adobe）公司的一款网页设计与开发软件。Adobe DNG Software Development Kit (SDK)是美国奥多比（Adobe）公司推出的一款软件开发工具包。本周，上述产品被披露存在多个漏洞，攻击者可利用漏洞获取敏感信息，在系统上执行任意代码，导致拒绝服务。  
  
CNVD收录的相关漏洞包括：Adobe InDesign越界读取漏洞（CNVD-2026-11767）、Adobe InDesign堆缓冲区溢出漏洞（CNVD-2026-11770）、Adobe InDesign缓冲区溢出漏洞（CNVD-2026-11772、CNVD-2026-11773）、Adobe Dreamweaver Desktop输入验证错误漏洞（CNVD-2026-11774）、Adobe DNG Software Development Kit (SDK)越界读取漏洞、Adobe DNG Software Development Kit (SDK)输入验证错误漏洞、Adobe DNG Software Development Kit (SDK)拒绝服务漏洞。其中，除“Adobe InDesign越界读取漏洞（CNVD-2026-11767）、Adobe DNG Software Development Kit (SDK)越界读取漏洞、Adobe DNG Software Development Kit (SDK)拒绝服务漏洞”外其余漏洞的综合评级为“高危”。目前，厂商已经发布了上述漏洞的修补程序。CNVD提醒用户及时下载补丁更新，避免引发漏洞相关的网络安全事件。
